# PalindroneCheckerApp
Objective:
To develop a Java program that checks whether a given string or number is a palindrome.

-->Take Input

Accept input from the user (String or Number).

Store the input value.


-->Process Input

Reverse the string/number

-->Compare

Compare original value with reversed value.

-->Display Result

If both are same → Print “Palindrome”

Else → Print “Not a Palindrome”
